# Puissance 4 (Terminal)

Petit jeu Puissance 4 en terminal, avec mode bot vs bot ou humain vs bot.

## Prerequis
- Python 3.6+

## Installation rapide
Ouvrir un terminal, exécuter : 
```
git clone https://github.com/AjiLI-meow/puissance-4.git
cd puissance-4
```

## Lancer le jeu
Exécuter :
```
python3 main.py
```
